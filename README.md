# luke_test

## How to start

#### Project hirerarchy
- |-- luke_test
   - |-- client
   - |-- server

### How to run server

- `cd server && npm i && npm start`

OR

- Spin up a `Docker container at -p 5023:5023`


### How to run client

- Make sure you have `node` installed.
- Clone the repository and run `cd client && yarn && yarn start`
